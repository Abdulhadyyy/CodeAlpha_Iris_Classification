# 🌸 Iris Flower Classification - CodeAlpha Task 1

### 📊 Project Overview
This repository contains my submission for **Task 1** at **CodeAlpha**. The goal was to classify Iris flowers into three species based on petal and sepal measurements.

### 🔍 Project Explanation (As per PDF Requirements)
1. **Data Cleaning:** I started by loading the `Iris.csv` dataset and dropped the 'Id' column, as it was not necessary for model training.
2. **Exploratory Data Analysis (EDA):** I used **Seaborn's Pairplots** to visualize how features like petal length and width separate the species. I also used a **Correlation Heatmap** to see the relationship between different variables.
3. **Model Selection:** I compared three algorithms: 
   - **Logistic Regression**
   - **Random Forest**
   - **SVM (Support Vector Machine)**
4. **Conclusion:** After evaluation, the **SVM** model provided the best results with **100% accuracy**, perfectly classifying all test samples.

### 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Pandas, NumPy, Seaborn, Matplotlib, Scikit-Learn
